# warmy-micropython


An IOT thermostat based on micropython, esp8266 board and DS1820 thermometer.


```json
{
  "broker": "<yourBroker>",
  "port": 1883,
  "sensor_pin": 0,
  "client_id": "<warmyId>",
  "topic": "<mainTopic>",
  "name": "<warmyName>",
  "essid": "<yourSSID>",
  "pwd": "<yourPassword>"
}
```
